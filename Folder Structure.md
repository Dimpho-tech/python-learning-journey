# Lesson 01: Introduction to Python
## What is Python?

Python is a high-level programming language that is easy to learn and widely used for:

- Web development
- Data Science
- Artificial Intelligence
- Automation
- Cybersecurity
- Software Development

---

## Why Learn Python?

- Easy to read
- Beginner friendly
- Large community
- Cross-platform
- Thousands of libraries

---

## Python Syntax

Python uses indentation instead of curly braces.

Example:

```python
print("Hello World")
```
